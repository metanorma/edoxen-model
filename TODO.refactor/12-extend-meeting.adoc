= 12 — Extend Meeting entity

== Why

The Meeting entity needs all the new fields from the broadened
scope. This TODO consolidates the Meeting-level changes.

== File to modify

- `models/meeting.lutaml`

== Current state (to be replaced)

```
class Meeting {
  identifier, urn, ordinal, type, status, year,
  date_range, committee, committee_group,
  venues (Location[]), general_area, city, country_code,
  virtual (Boolean), chair, secretary, host, hosts,
  source_urls, landing_url, registration_url,
  agenda, schedule, deadlines,
  attendance, vote_records, minutes,
  localizations, relations, resolution_refs
}
```

== New state

```lutaml
class Meeting {
  # Identity
  identifier: StructuredIdentifier[1..*]
  urn: String
  ordinal: Integer
  series_ref: String[0..1]              # URN → MeetingSeries

  # Classification
  type: MeetingType
  status: MeetingStatus
  visibility: Visibility                # NEW (from iCalendar CLASS)

  # Time
  date_range: DateRange
  recurrence: Recurrence[0..1]          # NEW (ISO 8601-2 §13 structured)

  # Place (polymorphic, multi)
  venues: Venue[0..*]                   # REPLACES old venues (Location) + virtual (Boolean)
  general_area: String                  # human-readable area name
  city: String                          # convenience (derivable from venues[].unlocode)
  country_code: String                  # convenience

  # Organization
  committee: String
  committee_group: String
  officers: Officer[0..*]               # NEW (replaces chair, secretary)
  hosts: HostRef[0..*]                  # REPLACES host + hosts

  # Documents
  source_urls: SourceUrl[0..*]
  landing_url: String
  registration_url: String

  # Structure
  agenda: Agenda
  components: MeetingComponent[0..*]    # NEW (replaces schedule)
  deadlines: Deadline[0..*]

  # People
  attendance: Attendance[0..*]

  # Records
  minutes: Minutes[0..*]

  # Outcomes (canonical location)
  decisions: Decision[0..*]             # RENAMED from resolution_refs
  motions: Motion[0..*]                 # NEW
  votings: Voting[0..*]                 # NEW (replaces vote_records — Voting holds them)

  # Localization
  localizations: MeetingLocalization[0..*]

  # Relations
  relations: MeetingRelation[0..*]

  # Extensions
  extensions: MeetingExtension[0..*]
}
```

== Fields REMOVED

- `year` — derivable from `date_range.start`
- `virtual: Boolean` — replaced by polymorphic `venues[]`
- `chair: Person` — moved to `officers[]` with role
- `secretary: Person` — moved to `officers[]` with role
- `host: String` — singular dropped, use `hosts[]`
- `schedule: ScheduleItem[]` — replaced by `components[]`
- `vote_records: VoteRecord[]` — moved inside `votings[].vote_records`

== Fields ADDED

- `series_ref` — URN to MeetingSeries
- `visibility` — public/private/confidential
- `recurrence` — structured Recurrence
- `officers` — replaces chair/secretary
- `components` — replaces schedule, adds sub-events
- `decisions` — renamed from `resolution_refs` (now holds Decision refs)
- `motions` — new canonical location
- `votings` — new canonical location
- `extensions` — profile mechanism
