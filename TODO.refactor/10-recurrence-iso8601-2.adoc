= 10 — Recurrence: structured ISO 8601-2 §13

== Why

User (2026-07-03): "queryable. yes."

Recurrence must be QUERYABLE — not a free-form ISO 8601-2 string.
A structured `Recurrence` class that mirrors ISO 8601-2 §13
syntactically, with `to_iso8601_2` / `from_iso8601_2` round-trip
methods on the gem side (NOT hand-rolled `to_h` — a structural
adapter, declared via lutaml-model mapping).

== Files to create

- `models/recurrence.lutaml`
- `models/recurrence_freq.lutaml`
- `models/recurrence_by_day.lutaml` (for BYDAY parts)
- `references/iso8601-2.adoc` (includes ISO 8601-2 §13 verbatim)

== Schema

```lutaml
class Recurrence {
  freq: RecurrenceFreq                 # SECONDLY | MINUTELY | HOURLY | DAILY |
                                        # WEEKLY | MONTHLY | YEARLY
  interval: Integer                    # default 1
  count: Integer[0..1]                 # number of occurrences
  until: DateTime[0..1]                # end date
  by_day: RecurrenceByDay[0..*]        # weekday + optional ordinal (e.g. -1MO)
  by_month_day: Integer[0..*]          # 1..31
  by_month: Integer[0..*]              # 1..12
  by_week_no: Integer[0..*]            # 1..53
  by_year_day: Integer[0..*]           # 1..366
  by_hour: Integer[0..*]               # 0..23
  by_minute: Integer[0..*]             # 0..59
  by_second: Integer[0..*]             # 0..59
  by_set_pos: Integer[0..*]            # e.g. -1 for "last"
  week_start: String[0..1]             # MO | TU | ... (default MO)
  extensions: MeetingExtension[0..*]
}

class RecurrenceByDay {
  ordinal: Integer[0..1]                # null=every; +1=first; -1=last
  weekday: String                       # MO | TU | WE | TH | FR | SA | SU
}

enum RecurrenceFreq {
  secondly, minutely, hourly, daily,
  weekly, monthly, yearly
}
```
