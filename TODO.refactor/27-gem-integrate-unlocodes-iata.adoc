== Integration pattern

```ruby
module Edoxen
  class VenueValidator
    def initialize(venue)
      @venue = venue
    end

    def validate
      case @venue.kind
      when VenueKind::PHYSICAL
        validate_physical(@venue.physical)
      when VenueKind::VIRTUAL
        validate_virtual(@venue.virtual)
      end
    end

    private

    def validate_physical(details)
      # Optional: validate UNLOCODE if present
      if details.unlocode && !Unlocodes.find_by_code(details.unlocode)
        errors << "Unknown UN/LOCODE: #{details.unlocode}"
      end
      # Optional: validate IATA code if present
      if details.iata_code && !Iata.find_by_code(details.iata_code)
        errors << "Unknown IATA code: #{details.iata_code}"
      end
    end
  end
end
```

== Auto-population

```ruby
if details.unlocode && details.city.nil?
  entry = Unlocodes.find_by_code(details.unlocode)
  details.city = entry&.name
end
```

Same for IATA → airport name and coordinates.
